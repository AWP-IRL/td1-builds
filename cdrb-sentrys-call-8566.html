Build Guide

This build has a very good dps potential, and still has some sustainability. I like to run Urban MDR with Destructive/Accurate, why accurate thats a personal thing i think it helps u to improve ur shots, i use it for quick spamming and fast focus the shots land more consistendly while spamming, and for that i run stability on backpack as well just to prevent bouncing effect. My goal is aiming as consistend as i can with enough survivability even while high fa spected. I hope u like this build but u can always adapt it to ur preference.

For faster dot spam some use the pistol in most cases for pve builds im using a custom pf 45 thats a blueprint with gs 204, to have a better acces at talent requiremends. Talent Toxic can definatly help because it can blind NPC wich perfectly combines with Coolheaded quicker cooldowns on the ultimate and skills, i tend to use the pistol if im in need of the ultimate to farm it back.

The M700C is good for damage if there marked with 6 dots an ocassional crit can go up like 4,5 million damage
